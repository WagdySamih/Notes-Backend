# Notes-Backend

## Available Scripts

### `npm run start`
### `npm run dev`


## This App support:
* Create Notes
* Get All notes with search provided
* Upadet Note
* Delete Note


## API endpoints:

>GET /notes 

>POST /notes

>delete /notes/:noteId

>PATCH /notes/:noteId

## Dependecies:
* Express
* mongodb
* mongoose
* joi
* express-async-handler
* cors
